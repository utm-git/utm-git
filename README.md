# Utkarsh Mehrotra 👋
Building systems at the **intersection of applied ML and distributed architectures**.

I architect highly scalable, fault-tolerant backend platforms. At Samsung, I designed consistency-critical systems for lending at scale. Now exploring how distributed system principles apply to AI — autonomous agents, knowledge layers, and decision systems that require the same rigor as financial platforms.

### 🚀 Current focus
- **ML-aware backend architectures**: Systems that serve autonomous agents with consistency guarantees, observability, and recovery patterns
- **Self-evolving knowledge systems**: RAG pipelines, continuous learning, and context refinement with exactly-once semantics
- **Agent orchestration & reliability**: Multi-agent coordination with the same rigor as distributed transactions
- **Production ML infrastructure**: Feature pipelines, model serving, and inference orchestration built on proven distributed patterns
- **Real-time AI-driven systems at scale**: High-throughput, low-latency decision engines where ML shapes system behavior — combining event streams, vector stores, and autonomous agents to handle planet-scale workloads with sub-100ms latencies and zero data loss

### 💡 What I've built
Globally distributed notification platforms, payment orchestration systems, event sourcing architectures, operational maturity patterns (chaos engineering, distributed tracing) — now applying these to AI systems

### 🛠️ Stack
**Languages**: Java, Python, JavaScript/TypeScript | **Expertise**: Distributed systems, Event-driven architecture, Microservices, Consensus algorithms, AWS/Terraform, Kafka, PostgreSQL, Redis
